# 3-Operators
The third lesson deals with operators.

There are 5 operators:
        <br>    ``+`` (addition)
        <br>    ``-`` (subtraction)
        <br>    ``*`` (multiplication)
        <br>    ``/`` (division) -> 5 / 2 = 2
        <br>    ``%`` (modulo) -> 5 % 2 = 1

When you want to add a value to the initial variable, you do this:
       <br>    ``var = var + 1``

/!\ Much simpler
<br>We use ``i++`` or ``i += X``

Ex:
        <br>    ``var = var + X -- > var += X``
        <br>``A = A - X``
        <br>``A -= X``
        <br>``A += 1 --> A = A + 1``
        <br>``A -= 1 --> A = A - 1``
        <br>``variable = i++``
        <br>``variable = ++i``
