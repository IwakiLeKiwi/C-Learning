# 5-Loops
The fifth lesson is about loops.

There are 3 types of loops:
        ``while ()``
        ``do {
        } while ()``
        ``for () {
        }``

``for ()`` takes 3 parameters 
1 -> variable initialization
2 -> condition (like while())
3 -> what happens (u++ -> add 1 to u value)
